Tabs os-002

Also known as: Tab bar, Tab group, Segmented control

Switch between related content sections.

Example
underline (default)
Primary blue #3b82f6, indigo #a5b4fc, violet #ddd6fe. Five gradient stops create the aurora colour field.
Base opacity 0.5. Mask extends from 10% to 70%. Adjustable per instance.
60s linear loop. Background position shifts from 50% to 350%. Respects prefers-reduced-motion.
Variants
underline
pill
How it works
One tab is active at a time. Clicking a tab shows its panel and hides the others. The active tab is visually distinguished — underline variant uses a bottom border accent, pill variant uses a filled background. Keyboard accessible: arrow keys move between tabs, Enter/Space activates.
Relationships
None — standalone component. Commonly used inside cards, settings panels, and design system pages.
